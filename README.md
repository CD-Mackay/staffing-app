# Welcome to the Staff Management Application
   This React App is used to handle internal staffing an team management. Employee profiles contain their department, direct superior as well as relevent skills associated with each employee. The list can be filtered by any of the available metrics making finding the right employee easy. 
    - Managers have the option to add new qualifications to an employee.
